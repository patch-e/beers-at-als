Views
Tap a viewing style of your choice. The beers will automatically layout according to your selection. Viewing styles can be bookmarked directly for your convenience.

  • Table - the beers in tabular format.
    Tap column headings to sort. Smaller screens can swipe the table left and right to scroll horizontally.
  • List - the beers in vertical list format.
  • Block - the beers in block flow format.
    Larger screens can display more blocks horizontally at a time.

Filtering
Enter filter criteria in any of the beer lists. All fields are immediately filtered upon as you type. For example, type IPA and any beer name or style containing "IPA" will be found.

About

Made by Patrick Crager, lover of beer and fan of Al's of Hampden / Pizza Boy Brewing Co. in Enola, PA. If you have never had the pleasure of visiting Al's, make it a priority and use my app while you are there :)

I made the app because Al's website wasn't mobile friendly and I was constantly not wanting to leave my table to see the tap list which isn't easily readable from every seat in the building! In other words, pure physical laziness and an excuse to build something fun.

Technical Crap
The front end of this website is an AngularJS app utilizing Bootstrap and jQuery for the presentation. The back end, that provides the live up-to-date data from Al's website, is a node.js app running on IIS 7.5. The code is open source and available on github.
